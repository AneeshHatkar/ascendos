# ascendOS

ascendOS is a modular, SQL-backed, AI-powered personal operating system for proof, execution, career, learning, research, health, body, life admin, creativity, grimoire-to-action translation, memory, analytics, and total-life dashboards.

The AI companion inside ascendOS is Carnos.
