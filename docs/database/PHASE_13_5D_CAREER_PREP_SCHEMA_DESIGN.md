# Phase 13.5D Career Prep Schema Design

Status: Repair scope.

## Purpose

Phase 13.5D closes the missing career-prep layer identified after Phases 1-13.

Already implemented by Phase 8:

- `job_applications`
- `job_application_events`
- `networking_contacts`
- `networking_interactions`
- `job_referrals`
- `resume_versions`
- `resume_bullets`
- `interviews`

Added by Phase 13.5D:

- `behavioral_stories`
- `question_bank`
- `mock_interviews`
- `resume_usage`

## Boundary

This phase does not implement:

- AI answer generation
- autonomous interview feedback
- resume rewriting
- automatic applications
- outreach emails
- scraping
- background jobs
- voice
- memory/RAG
- web search
- analytics
- settings/privacy
