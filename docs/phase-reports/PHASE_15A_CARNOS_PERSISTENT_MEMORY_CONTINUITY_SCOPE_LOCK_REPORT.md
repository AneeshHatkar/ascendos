## Protected safety contract

Carnos should know what is going on every time the user returns, but must never silently store, misuse, expose, distort, over-trust, or hide memory behavior.

Memory must be:

- User-controlled.
- Inspectable.
- Reversible.
- Provenance-linked.
- Privacy-aware.
- Sensitivity-aware.
- Conflict-aware.
- Staleness-aware.
- Auditable.
